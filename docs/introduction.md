# MoraMapGen

MoraMapGen is a procedural dungeon and corridor generator for Unity,
designed for roguelikes and modular level workflows.

## Who is this for?
- Roguelike and dungeon crawler developers
- Procedural level generation
- Editor-time and runtime generation
