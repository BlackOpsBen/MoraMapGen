# Quick-Start Setup
#### 1. Add the MoraMapGen component to an empty GameObject in your scene
![MoraMapGen Component](../images/quick-start/AddComponent.png)

#### 2. Click "New" to create a new MapGenSettings asset
![New Settings](../images/quick-start/New_Settings_Asset_Button.png)

#### 3. Click the Area Options picker
![Area Options Picker](../images/quick-start/Area_Options_Picker.png)

#### 4. Select an existing Area Options asset
![Demo Area Options asset](../images/quick-start/AreaOptions.png)

#### 5. Click the "Generate Map" button
![Generate Map button](../images/quick-start/Generate_Map_Button.png)

Alternatively you can turn on "Generate On Awake" and then press Play to generate at runtime:
![Generate on Awake toggle](../images/quick-start/Generate_On_Awake.png)
